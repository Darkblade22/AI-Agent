# 🤖 Личный ИИ-агент в Telegram

Telegram-бот на базе Claude (Anthropic). Умеет:

- 💬 общаться и помнить контекст диалога;
- 🌐 искать актуальную информацию в интернете;
- 🧠 запоминать факты о тебе между перезапусками;
- 📝 создавать и читать заметки (файлы);
- ⏰ ставить напоминания.

---

## Шаг 1. Получи два ключа

### 1.1. Токен Telegram-бота
1. Открой в Telegram бота [@BotFather](https://t.me/BotFather).
2. Отправь команду `/newbot`.
3. Придумай имя и username бота.
4. BotFather пришлёт строку вида `123456:ABC-DEF...` — это токен.

### 1.2. API-ключ Anthropic
1. Зайди на [console.anthropic.com](https://console.anthropic.com).
2. Пополни баланс (хватит $5 для старта) в разделе **Billing**.
3. В разделе **API Keys** создай ключ — строка вида `sk-ant-...`.

### 1.3. Свой Telegram ID
Напиши боту [@userinfobot](https://t.me/userinfobot) — он пришлёт твой числовой ID.

## Шаг 2. Настрой проект

В терминале перейди в папку проекта:

```bash
cd "/Users/danilstennikov/Documents/CODING/01. MY PROJECT/AI AGENT"
```

Создай файл настроек из шаблона и впиши свои значения:

```bash
cp .env.example .env
open -e .env   # откроется в редакторе — впиши токены и сохрани
```

В `.env` нужно заполнить:
- `TELEGRAM_BOT_TOKEN` — токен из BotFather
- `ANTHROPIC_API_KEY` — ключ из console.anthropic.com
- `ALLOWED_USER_IDS` — твой Telegram ID

---

## Шаг 3. Установи зависимости и запусти

Один раз создай виртуальное окружение и поставь библиотеки:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Запусти бота:

```bash
python bot.py
```

Если всё верно, увидишь: `✅ Бот запущен`. Открой своего бота в Telegram,
напиши `/start` и начинай общаться.

> Бот работает, пока запущен этот процесс и включён Mac.
> Останови его клавишами **Ctrl+C**. В следующий раз достаточно:
> `source .venv/bin/activate && python bot.py`

## Что где лежит

| Файл | Назначение |
|------|------------|
| `bot.py` | запуск, Telegram, напоминания |
| `agent.py` | общение с Claude и обработка инструментов |
| `tools.py` | инструменты: память, файлы, напоминания |
| `memory.py` | долговременная память (факты о тебе) |
| `config.py` | загрузка настроек из `.env` |
| `data/` | память и напоминания (создаётся автоматически) |
| `workspace/` | заметки/файлы бота (создаётся автоматически) |

## Частые вопросы

**Бот отвечает «отвечаю только своему владельцу».**
В `.env` неверный `ALLOWED_USER_IDS`. Впиши свой ID от @userinfobot.

**Хочу более умные ответы.**
В `.env` поменяй `CLAUDE_MODEL=claude-opus-4-8` (дороже, но мощнее).

**Хочу, чтобы работал 24/7 без включённого Mac.**
Нужно разместить бота на сервере (Railway, VPS и т.п.) — это следующий шаг,
напиши, и настроим.
